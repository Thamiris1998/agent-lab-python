# 🎮 Soc Ops - Social Bingo

**Break the ice at any mixer!** A dynamic bingo game where players find people matching questions and complete 5 in a row. Built with cutting-edge AI agents and modern web tech.

> Perfect for in-person events, networking sessions, and team building activities.

---

## ✨ What You'll Build

This **hands-on AI development lab** teaches you to:
- 🤖 Orchestrate **multi-agent workflows** with GitHub Copilot
- 🎨 Build **responsive frontends** with HTMX (no JavaScript frameworks!)
- 🚀 Design **AI-powered backends** with FastAPI
- 🧪 Master **test-driven development** and context engineering

Not just theory—you'll have a **working game** that's fun to play!

---

## 🚄 Quick Start

```bash
# 1. Install dependencies
uv sync

# 2. Run tests
uv run pytest

# 3. Start the dev server
uv run uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

# 4. Open in browser
open http://localhost:8000
```

- **Testing**: pytest + comprehensive coverage
